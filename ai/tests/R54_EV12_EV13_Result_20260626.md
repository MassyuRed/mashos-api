## 4. EV13 実装内容

EV13では、EV10 rating rows と EV12 question need observation rows の整合性を body-free に検査する guard を追加した。

検出する整合性問題:

```text
RED / REPAIR_REQUIREDなのに no_question_needed としている
P5 surface repair requiredなのに P8 design material candidateへ昇格している
PASSなのに not_question_*_repair_required が残っている
insufficient materialなのにPASS扱い、またはexecution blockerがない
rating rows と question rows の case_ref set が一致しない
```

EV13で守ること:

```text
P5 repair対象をP8材料へ逃がさない
not_question_*_repair_required を P8 material candidate にしない
consistency issueがある場合はEV14へ進めない
actual review completion claimはしない
disposal verifiedにはしない
P6/P8/releaseはfalseのまま保持する
```
